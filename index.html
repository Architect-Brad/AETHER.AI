Overview

Agent Mode v1 ★ NEW

Click AGENT in the toolbar. Enter a goal, set step limit (1–30) and approval mode (auto or per-step). AETHER runs autonomously using tools until the goal is complete or limits are reached.

How it works

  • Plan: AI produces a structured <agent:plan> of numbered steps
  • Act: Executes one tool call per step using [[tool: arg]] syntax
  • Evaluate: After each tool result, re-assesses progress and updates plan
  • Retry: If a step fails, tries an alternative approach autonomously
  • Complete: Outputs <agent:done>summary</agent:done> when goal is achieved

Configuration options

OptionValuesDefault
Max Steps1–3010
Tool Budget1–5020
Approval ModeAuto (run all) / Per StepAuto
Cost note: Each step is a separate API call. A 10-step agent task = 10+ API calls. Use local models (Qwen3.5) for cost-free agent runs.

# Minimal server file (aether_browser_agent.py)
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from pydantic import BaseModel

app = FastAPI()
app.add_middleware(CORSMiddleware, allow_origins=["*"],
    allow_methods=["*"], allow_headers=["*"])

class Req(BaseModel):
    task: str
    engine: str = "browser-use"

@app.post("/run")
async def run(req: Req):
    # Add your browser-use or playwright logic here
    return {"result": f"Executed: {req.task}"}

# Start the server
uvicorn aether_browser_agent:app --port 7860 --reload
